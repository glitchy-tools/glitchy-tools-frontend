# Glitchy Tools

Internal tools platform. Vue 3 + TypeScript + Tailwind. Hosted on `glitchy.tool`, embedded via iframe in the main Glitchy app.

## Setup

```bash
npm install
npm run dev
```

## Adding a Tool

Create `src/tools/your-tool-name/index.vue` — it auto-registers.

## Current Tools

- Watermark Remover
- Notification Mockup
- Cloaker
- Email HTML Generator
- Trending Angles
- SMS Verification
